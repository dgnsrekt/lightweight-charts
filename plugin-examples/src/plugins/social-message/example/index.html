Social Message

Hint: Click messages to open platform URLs. Switch between platform themes to see different styles. Toggle between fixed (anchored to time/price) and draggable positioning modes.

The Social Message plugin displays social media message annotations on charts from multiple platforms (Discord, X/Twitter, Telegram), similar to TradingView's social annotation features. Each platform has its own distinct theme, branding, and special features like verified badges for X/Twitter.

Features

  • Multi-Platform Support: Discord, X.com (Twitter), and Telegram with platform-specific styling
  • Platform Switcher: Dynamically change themes to match different social media platforms
  • Verified Badges: X/Twitter messages can display blue verified checkmarks
  • Dual Positioning Modes: Fixed (time/price anchored) or draggable
  • Crosshair Lines: Visual connection between message and chart position
  • Interactive: Click to open original platform URLs
  • Extensible: Easy to add new platforms via adapter interface
